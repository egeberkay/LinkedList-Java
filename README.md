# Custom Linked List Implementation

This project is a Linked List implementation written from scratch to reinforce the fundamentals of data structures and algorithm logic.

## 🚀 Features

The current structure supports the following basic Linked List operations:

* Adding a new node
* Deleting a specific value from the list
* Printing list elements to the console
* [Inserting an element in the middle, delete an element or all elements, bublesort , Stack and Queue]

## 💻 Technologies Used

* **Language:** Java
* **Environment:** Java Development Kit (JDK)
